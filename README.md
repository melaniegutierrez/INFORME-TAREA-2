### 5. CONCLUSIONES

Se puede concluir que conocer la ley de ohm es muy importante para un estudiante de electronica, ya que nos permite conocer las bases teóricas de las medidas y la relación entre voltaje resistencia y corriente, de tal manera tenemos un mejor concepto de como se diseña un circuito y las cantidades correctas de energia de cada elemento para que el circuito funcione bien y no tenga fallos. 

La potencia y la eficiencia que puede tener una fuente nos ayudara a manejar de mejor manera la cantidad de energia y electricidad con la que estamos trabajndo, ademas de aprovecharla de mejor manera para cuidar el circuito de posibles sobrecalentamientos y daños irreversibles en los elementos del circuito.
